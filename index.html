Desarrollo de Software

Objetivo General

• Identificar las etapas de desarrollo de un Software y las necesidades en el campo de la programación de sistemas, para la solución de problemas, mediante el análisis de requerimientos y sus diferentes modelos.


Objetivos del OVI

1- Informar sobre el desarrollo de Software y sus conceptos basicos.

2- Conocer mas a fondo sobre las herramientas y formas de desarrollar Software.


Conclusión

El desarrollo de software también conocido como el ciclo de la vida, es una estructura diseñada para la creación de un software mediante diferentes modelos.

Procesos para el desarrollo del software

Son un conjunto de actividades utilizadas para el correcto desarrollo de un sistema de software, como son:
Planificación, Diseño, Implementación, Pruebas y Mantenimiento.
